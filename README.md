# README

## Installation

- Clone repo
- npm i
- Configure .env

## Config .env

HOST = "" // SMTP host

PORT = 993 // Could be other for you

SECURE = "true"

USER = "email@adress.com"

PASS = "password"

## Run

2 instances:
- node src/server.js // Email listener
- npm run dev

## Page
http://localhost:5173/flow
